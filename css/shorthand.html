Shorthand properties

Background

Valores globais

  • inherit
  • initial
  • revert
  • revert-layer
  • unset
  • background-attachment
    • scroll: fixado em relação ao elemento em si e não scrolla com o conteúdo
    • fixed: fixado em relação ao viewport
    • local: fixado em relação aos elementos do conteúdo
  • background-clip
    • border-box: o background se estende para a borda externa
    • padding-box: o background se estende para o padding externo
    • content-box: o background é clippado ao conteúdo da caixa
    • text: o background é clippado ao texto do primeiro plano
  • background-color
    • usando palavra-chave, por exemplo 'blue' ou 'transparent'
    • usando RGB - sistema de coordenadas cúbicas: rgb() ou rgba()
    • usando HSL - sistema de coordenadas cilíndricas: hsl() ou hsla()
    • usando HWB - sistema de coordenadas cilíndricas: hwb()
    • usando LCH - sistema de coordenadas cilíndricas
    • sistema de coordenada Lab: lab()
    • seleção de cor: color()
  • background-image
    • none:
    • image:
  • background-origin
  • background-position
  • background-repeat
  • background-size
